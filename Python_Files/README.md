#  Transport Emissions Dashboard

Interactive Streamlit dashboard analyzing global greenhouse gas emissions from the transport sector (1990–2021).

## About

This project was developed for the *5DATA004C Data Science Project Lifecycle* module at the University of Westminster. It presents insights into transport-sector GHG emissions aimed at policymakers, finance professionals, and technology experts attending a global sustainability conference.

##  Dataset

*WRI Climate Watch — Greenhouse Gas Emissions (Transport)*
Source: [World Bank Data360](https://data360.worldbank.org/en/indicator/WRI_CLIMATEWATCH_ALL_GHG_TRANSPORT)


## Tech Stack

- Python 3.11+
- Streamlit
- Pandas, NumPy
- Plotly, Matplotlib, Seaborn
- Scikit-learn
